Tmux — Final 3-Theme Design

Same shell content rendered on all three themes. Only the bar changes.

About the shell prompt: in real tmux, your shell prompt (the ❯ ~/code/tmux on main line) is drawn by zsh/Powerlevel10k, not by tmux. So switching tmux themes does not change how it looks — it's identical regardless. My earlier mockups tinted it differently per theme, which was misleading. Below, all three terminals show byte-for-byte identical shell text (same words, same colors). Only the bar differs.

Tweaks applied vs the recommendation:
• Prefix indicator added to all 3 (the small slot at the very left of the bar — dim dot when idle, lit when C-a is held)
• Live CPU sparkline removed from tmux2k — replaced with static CPU 38% so nothing in your peripheral vision is moving while you code
