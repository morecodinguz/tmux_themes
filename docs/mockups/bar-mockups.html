Tmux Bar Redesigns

Three concrete designs. Rendered 1:1 with the catppuccin mocha hex codes you'd put in tmux config. Each theme keeps its locked-in temperature identity and adds real structural character.

What's new vs current:
  • Directory pill in sapphire — adds context you can't currently see at a glance
  • Git ahead/dirty counts (↑2 ●3) replace plain branch text

  • Active window gets a ● mark + lavender; inactive stays surface2 dim

base#1e1e2e mauve#cba6f7 lavender#b4befe peach#fab387 yellow#f9e2af pink#f5c2e7 subtext0#a6adc8 overlay0#6c7086
What's new vs current:
  • Only the session keeps a pill — mauve, fully rounded ends. It's the single focal anchor.
  • Everything else is flat text with subtle middle-dot separators. No more peach/yellow/pink pills competing for attention.
  • Window list: 1 · zsh typographic format — number, dim dot, name. Lavender for active, overlay0 for inactive.
  • Right side reads like one sentence: ~/code/tmux · main · 82% · 10·59
  • Time uses · instead of : — small flourish that fits the spacious mood
  • Tons of breathing room — the bar feels designed, not packed

What's new vs current:
  • Solid block frames ▌ ▐ around the session — looks like a hot-stamp on the bar, very print/poster
  • Window items use thin half-block ▎ as typographic divider (not a separator pill)
  • Branch in bracketed caps [ MAIN ] — terminal/console aesthetic, only red used as highlight
  • No icons on right side — battery is just 82%, time is text, date in caps with · separators
  • Pure black canvas amplifies warm accents (rosewater frame, peach battery, red [brackets])
  • Asymmetric weight: heavy left, sparse pipe-separated right
