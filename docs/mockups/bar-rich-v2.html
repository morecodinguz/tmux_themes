Rich v2 — same structural language across all 3

Glacier is unchanged (you liked it). Atelier and Bauhaus now use the same parallel structure: cascading powerline pills for session/path/branch, dot-marker window list, framed stat group, separate time pill. Each theme expresses that structure in its own temperature palette.

Same skeleton, three temperatures:
Glacier (cool) — lavender → sapphire → green cascade
Atelier (pastel) — mauve → lavender → peach cascade
Bauhaus (warm) — rosewater → peach → red cascade
